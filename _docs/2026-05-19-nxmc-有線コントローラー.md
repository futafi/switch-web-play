# nxmc 有線コントローラー (2026-05-19)

nuxbt（Bluetooth）が Authentication Failure で使えなかったため、有線USB UART経由でSwitchを操作する方式に切り替えた。

## 構成

```
Browser ──WebSocket──> Go Server ──Serial(115200)──> Arduino Leonardo ──USB HID──> Switch
                         :8080        /dev/ttyUSB0     (FTDI FT232R)
```

- **ハードウェア**: PC <-> FTDI FT232R (USB UART) <-> Arduino Leonardo <-> Switch
- **ファームウェア**: Poke-Controller系。ボーレート115200に変更済み
- **シリアルデバイス**: `/dev/ttyUSB0` (FTDI FT232R USB UART A5069RR4)

## NX2プロトコル (0xAB)

11バイト固定長のバイナリパケット。ファームウェアはNX2モード時に `\n` をスキップせずバッファに蓄積し、11バイト到達で処理する。終端文字は不要。

| Byte | 内容 | 範囲 |
|------|------|------|
| 0 | ヘッダ `0xAB` | 固定 |
| 1 | ボタン下位バイト | bitmask |
| 2 | ボタン上位バイト | bitmask |
| 3 | Hat（十字キー） | 0-8 |
| 4 | 左スティックX | 0-255 (128=中央) |
| 5 | 左スティックY | 0-255 (128=中央) |
| 6 | 右スティックX | 0-255 (128=中央) |
| 7 | 右スティックY | 0-255 (128=中央) |
| 8 | キーボードモード | 0=なし, 1-5 |
| 9 | キーボードキー | キーコード |
| 10 | パディング | 0x00 |

### ボタンビットマスク

```
Y=0x0001  B=0x0002  A=0x0004  X=0x0008
L=0x0010  R=0x0020  ZL=0x0040 ZR=0x0080
-=0x0100  +=0x0200  LC=0x0400 RC=0x0800
HOME=0x1000  CAPTURE=0x2000
```

### Hat値

```
0=UP  1=UP_RIGHT  2=RIGHT  3=DOWN_RIGHT
4=DOWN  5=DOWN_LEFT  6=LEFT  7=UP_LEFT  8=CENTER
```

## ソフトウェア構成

```
nxmc/
  firmware/           Arduino参考コード
  go.mod
  protocol.go         NX2プロトコル定義・パケットビルダー
  controller.go       シリアル通信ラッパー
  cmd/
    nxmc-test/
      main.go         CLIテストツール（Aボタン押下確認用）
    nxmc-server/
      main.go         HTTP + WebSocket サーバー
      web/
        index.html    Gamepad API + キーボード入力 → WebSocket
```

### nxmc パッケージ（Go ライブラリ）

```go
ctrl, _ := nxmc.Open("/dev/ttyUSB0", 115200)
defer ctrl.Close()

r := nxmc.NewReport()
r.Buttons = nxmc.ButtonA
ctrl.SendReport(r)
```

### nxmc-server

- `:8080` で起動
- `GET /` — Web UI配信
- `GET /ws` — WebSocket（JSON形式のコントローラー状態を受信）

WebSocketメッセージ:

```json
{"buttons":4,"hat":8,"lx":128,"ly":128,"rx":128,"ry":128}
```

### Web フロントエンド

- Gamepad API: 60fpsポーリング、Standard Gamepad Mapping対応
- キーボード: WASD=左スティック、矢印=Hat、UIJKL=ボタン等
- 状態変化時のみWebSocket送信

## ビルド・実行

```bash
cd nxmc
go build ./cmd/nxmc-test && ./nxmc-test        # CLIテスト
go build ./cmd/nxmc-server && ./nxmc-server     # サーバー起動
```
